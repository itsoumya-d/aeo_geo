# Production Readiness Worklog

This folder contains a repo-grounded audit + refactor worklog for launch readiness:

- `route-inventory.md`: current routes, shipped screens, and known broken links.
- `technical-leaks.md`: user-facing technical/provider/infrastructure leakage to remove.
- `feature-trace.md`: trace each core feature from UI → service → Edge Function → DB.
- `text-overflow-fixes.md`: log of overflow/layout fixes (component → issue → CSS fix).
- `perf.md`: bundle/performance notes.
- `deployment-checklist.md`: env vars + Supabase/Stripe deployment checklist.
